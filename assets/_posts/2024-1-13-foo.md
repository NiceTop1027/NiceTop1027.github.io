# 백준 알고리즘 문제 풀이

## 1 백준ㅣ최댓값 2566번

```cpp
#include <iostream>
#include "../cTEST/stdc++.h"

using namespace std;

int main(){
    int arr[9][9];
    int hap,x,y;

    for(int i = 0; i < 9; i++){
        for(int j = 0; j < 9; j++){
            cin >> arr[i][j]; 
        }
    }

    for(int i = 0; i < 9; i++){
        for(int j = 0; j < 9; j++){
            if(hap <= arr[i][j]){
                hap = arr[i][j];
                x = i;
                y = j;
            }
        }
    }
    cout << hap << " " << x+1 << " " << y << endl;
    return 0;
}
```
이번 문제는 2차원 배열로 최댓값의 값, 행, 열 번호를 출력해야된다.

그러므로 처음에는 2차원 배열을 초기화 해주었다. 

그 다음 최댓값을 저장할 수 있는 **hap**변수와 행과 열을 저장할 수 있는 **x, y**를 만들었다.

그 후 이중 포문을 이용해 0부터 8까지인 총 9번을 반복하고 그 안에 똑같은 9번을 반복해서

```cpp
cin >> arr[i][j];
```
각 행과 열에 맞게 입력값을 받았다.


그런 후 최댓값과 최댓값의 행과 열을 표현하기 위해서 똑같이 이중 포문을 활용하였다.

똑같이 두 포문 모두 9번 반복하게 만들었고 **여기서가 중요하다.**

if 문을 활용해 
```cpp 
if(hap <= arr[i][j]){
  hap = arr[i][j];
  x = i;
  y = j;
```
hap이 만약에 arr[i][j]보다 작거나 같다면 hap에 arr[i][j]에 있는 값을 넣었고,

x 에는 그 최댓값이 위치한 행을 넣고, y 또한 최댓값이 위치해 있는 열을 넣었다.

이렇게 각 각 변수에 넣은 숫자들을 cout 해주면 완벽하게 출력할 수 있다.


## 2 백준ㅣ사분면 고르기 14681번

이번 문제는 매우 간단하다. 각 숫자를 입력받아 숫자에 맞게 사분면을 정하는 것이다.

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
	int x, y;
	cin >> x >> y;

	if (x > 0 and y > 0) {
		cout << "1" << endl;
	}

	else if (x < 0 and y > 0) {
		cout << "2" << endl;
	}

	else if (x < 0 and y < 0) {
		cout << "3" << endl;
	}

	else if (x > 0 and y < 0) {
		cout << "4" << endl;
	}
	return 0;
}
```

제일 중요한 부분은 if 문이다. 

일단 첫번째 if 문에서는..

x 가 만약 0보다 크면서 y 도 0보다 크면 **그것은 제1사분면인 것이다.**

똑같이 두번째 if 문에서는..

x 가 0보다 작으면서 y 가 0보다 크면 **그것은 제2사분면인 것이다!.**

이렇게 문제에 맞게 양수와 음수를 결정해주면 간단하게 문제를 풀 수 있을것이다.

## 3 백준ㅣ윤년 2753번

이번 문제는 윤년을 고르는 문제이다.

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
	int year = 0;
	cin >> year;
	
	if (year % 4 == 0 && !(year % 100 == 0) || year % 400 == 0) {
		cout << "1" << endl;
	}
	else {
		cout << "0" << endl;
	}
	return 0;
}
```

문제에서는 이렇게 나와있다.
```
연도가 주어졌을 때, 윤년이면 1, 아니면 0을 출력하는 프로그램을 작성하시오.

윤년은 연도가 4의 배수이면서, 100의 배수가 아닐 때 또는 400의 배수일 때이다.

예를 들어, 2012년은 4의 배수이면서 100의 배수가 아니라서 윤년이다. 1900년은 100의 배수이고 400의 배수는 아니기 때문에 윤년이 아니다. 하지만, 2000년은 400의 배수이기 때문에 윤년이다.
```
이렇게 알 수 있듯이 윤년은 연도가 4의 배수이면서, 100의 배수가 아닐 때 또는 400의 배수일 때이다.

윤년을 모르는 사람도 알 수 있듯이 굉장한 힌트였다..

이걸 이용해 코드를 짜면

```cpp
	if (year % 4 == 0 && !(year % 100 == 0) || year % 400 == 0)
```
이런식의 코드가 완성된다. 이렇게 만든 코드를 활용해 조건문을 완성하면 윤년인지 아닌지를 확인할 수 있는 프로그램을 만들 수 있다.


## 4 백준ㅣ알람시계 2885번

이번 문제는 시간 45분전에 알람을 맞쳐주는 프로그램을 만드는 알고리즘 문제이다.

```cpp
#include "../cTEST/stdc++.h"

using namespace std;

int main() {
    int H, M;
    
    cin >> H >> M;

    if (M >= 45) {

        cout << H << " " << M - 45;
    }   else {

        if (H == 0) {

            cout << 23 << " " << M + 15;
        } else {

            cout << H - 1 << " " << M + 15;
        }
    }
    return 0;
}
```
먼저 시간이랑 분을 초기화 해준다. **int H,M;**

그 다음 입력을 각각 받아주고.. 

if 문을 활용해서 만약 M이 45분보다 높거나 같으면 M-45를 하게 되어있다.. 왜 이렇게 코드를 짰는가??

```
보통 분은 총 60분이다. 그래서 만약 45분에에서 45분을 빼면 0분이므로.. 딱 맞쳐서 내려간다. 하지만 40분은 어떨까?
40분을 45분으로 빼면 다시 60분을 넘어 55분이 된다.. 이러므로 우리는 이걸 방지하기 위해 딱 45분하기 크거나 같으면으로 만들어서 값이 다시 되돌아가지 않도록 만들었다.
```

그런 후 만약 M 이 45분이 아닐때는 뭘까? 한마디로 **시** 부분이 -1 된다는것이다..

그러면 H-1를 이용해서 늦출 수 있다.

하지만 시 부분이 0일때는 문제가 발생한다. 0도 다시 돌아가야되지 않는가? 23시로..

그래서 어차피 0일때는 무조건 23이 될테니.. 그냥 출력할때의 값을 23으로 정했다.. 천재적이지 않은가?? ㅋㅋ

그런 후 분 계산은 조금 신기할 것이다.. 만약 분이 45보다 작을때는 그냥 그 값에 15을 더해주면 된다.. ㅋㅋ

그러면 정확한 분 계산을 알 수 있다.

**개인적으로 수학적인 느낌이 나서 굉장히 싫은 문제였다. 하지만 그 만큼 매우 흥미로웠던 문제였다.**


## 5 백준ㅣ구구단 2739번

이번 문제는 간단한 구구단 프로그램 만들기 문제였다.

```cpp
#include "../cTEST/stdc++.h"
//#include <bits/stdc++.h>

using namespace std;

int main() {
    int input;

    cin >> input;
    for(int i = 1; i < 10; i++){
        cout << input << " " << "*" << " " << i << " " << "=" << " " <<input*i << endl;
    }
    return 0;
}

```

간단하게 input에 입력값을 받고 그 값을 이용해 1~9까지의 포문을 만들어서 

1단에서 9단까의 전체적인 구구단을 표시했다.

딱히 설명할 필요가 있나 싶었다... 하지만 중요하니 하겠다.

```cpp
for(int i = 1; i < 10; i++)
```

보는거와 같이 for 문을 1~9인 값으로 설정한걸 알 수 있다.

그리고 밑쪽에 cout을 통해 

```cpp
cout << input << " " << "*" << " " << i << " " << "=" << " " <<input*i << endl;
```
구구단을 출력하는걸 볼 수 있다 솔직히 말해서 "  " 이거 부분이 개인적으로 지져분하긴 하지만 참고 했다.

결국 구구단을 쉽게 만드는법을 알려줬다. 쉽지 :)


## 6 백준ㅣ코딩은 체육과목 입니다 25314번

이번 알고리즘 문제도 간단한 문제였던것 같다.. (**개인적인 생각**)

```cpp
#include "../cTEST/stdc++.h"
//#include <bits/stdc++.h>

using namespace std;

int main() {
    int input;
    cin >> input;
    char test[5] = "long";

    for(int i = 0; i < input/4; i++){
        cout << test << " ";
    }
    cout << "int" << endl;
    return 0;
}
```
이번 문제는 간단하다 설명을 하자면 long이라는 문자열이 있는데 이게 int형 즉 4바이트에 맞게 문자열을 출력하도록 하면 된다.

글씨로는 힘드니 코드를 한번 확인해보자.. 

일단 cin << input; 으로 입력값을 받는다. 그 후 for 문을 활용해 i 가 입력값 나누기 4의 몫만큼의 반복을 하겠다는 소리인데.. 다들 이해하셨길 바란다..

대충 그 만큼 반복을 하면 cout으로 인해 그 수 만큼의 test 배열인 즉 문자열이 출력된다.

그래서 이걸 이용해 프로그램을 제작하였고 조건에 마지막에 int가 들어가서 int를 넣어줬다 그러면 완벽하게 프로그램을 완성할 수 있다!
