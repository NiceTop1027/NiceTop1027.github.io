# 백준 알고리즘 문제 풀이

## 1 백준ㅣ최댓값 2566번

```cpp
#include <iostream>
#include "../cTEST/stdc++.h"

using namespace std;

int main(){
    int arr[9][9];
    int hap,x,y;

    for(int i = 0; i < 9; i++){
        for(int j = 0; j < 9; j++){
            cin >> arr[i][j]; 
        }
    }

    for(int i = 0; i < 9; i++){
        for(int j = 0; j < 9; j++){
            if(hap <= arr[i][j]){
                hap = arr[i][j];
                x = i;
                y = j;
            }
        }
    }
    cout << hap << " " << x+1 << " " << y << endl;
    return 0;
}
```
이번 문제는 2차원 배열로 최댓값의 값, 행, 열 번호를 출력해야된다.

그러므로 처음에는 2차원 배열을 초기화 해주었다. 

그 다음 최댓값을 저장할 수 있는 **hap**변수와 행과 열을 저장할 수 있는 **x, y**를 만들었다.

그 후 이중 포문을 이용해 0부터 8까지인 총 9번을 반복하고 그 안에 똑같은 9번을 반복해서

```cpp
cin >> arr[i][j];
```
각 행과 열에 맞게 입력값을 받았다.


그런 후 최댓값과 최댓값의 행과 열을 표현하기 위해서 똑같이 이중 포문을 활용하였다.

똑같이 두 포문 모두 9번 반복하게 만들었고 **여기서가 중요하다.**

if 문을 활용해 
```cpp 
if(hap <= arr[i][j]){
  hap = arr[i][j];
  x = i;
  y = j;
```
hap이 만약에 arr[i][j]보다 작거나 같다면 hap에 arr[i][j]에 있는 값을 넣었고,

x 에는 그 최댓값이 위치한 행을 넣고, y 또한 최댓값이 위치해 있는 열을 넣었다.

이렇게 각 각 변수에 넣은 숫자들을 cout 해주면 완벽하게 출력할 수 있다.


## 2 백준ㅣ사분면 고르기 14681번

이번 문제는 매우 간단하다. 각 숫자를 입력받아 숫자에 맞게 사분면을 정하는 것이다.

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
	int x, y;
	cin >> x >> y;

	if (x > 0 and y > 0) {
		cout << "1" << endl;
	}

	else if (x < 0 and y > 0) {
		cout << "2" << endl;
	}

	else if (x < 0 and y < 0) {
		cout << "3" << endl;
	}

	else if (x > 0 and y < 0) {
		cout << "4" << endl;
	}
	return 0;
}
```

제일 중요한 부분은 if 문이다. 

일단 첫번째 if 문에서는..

x 가 만약 0보다 크면서 y 도 0보다 크면 **그것은 제1사분면인 것이다.**

똑같이 두번째 if 문에서는..

x 가 0보다 작으면서 y 가 0보다 크면 **그것은 제2사분면인 것이다!.**

이렇게 문제에 맞게 양수와 음수를 결정해주면 간단하게 문제를 풀 수 있을것이다.

## 3 백준ㅣ윤년 2753번

이번 문제는 윤년을 고르는 문제이다.

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
	int year = 0;
	cin >> year;
	
	if (year % 4 == 0 && !(year % 100 == 0) || year % 400 == 0) {
		cout << "1" << endl;
	}
	else {
		cout << "0" << endl;
	}
	return 0;
}
```

문제에서는 이렇게 나와있다.
```
연도가 주어졌을 때, 윤년이면 1, 아니면 0을 출력하는 프로그램을 작성하시오.

윤년은 연도가 4의 배수이면서, 100의 배수가 아닐 때 또는 400의 배수일 때이다.

예를 들어, 2012년은 4의 배수이면서 100의 배수가 아니라서 윤년이다. 1900년은 100의 배수이고 400의 배수는 아니기 때문에 윤년이 아니다. 하지만, 2000년은 400의 배수이기 때문에 윤년이다.
```
이렇게 알 수 있듯이 윤년은 연도가 4의 배수이면서, 100의 배수가 아닐 때 또는 400의 배수일 때이다.

윤년을 모르는 사람도 알 수 있듯이 굉장한 힌트였다..

이걸 이용해 코드를 짜면

```cpp
	if (year % 4 == 0 && !(year % 100 == 0) || year % 400 == 0)
```
이런식의 코드가 완성된다. 이렇게 만든 코드를 활용해 조건문을 완성하면 윤년인지 아닌지를 확인할 수 있는 프로그램을 만들 수 있다.


## 4 백준ㅣ알람시계 2885번

이번 문제는 시간 45분전에 알람을 맞쳐주는 프로그램을 만드는 알고리즘 문제이다.

```cpp
#include "../cTEST/stdc++.h"

using namespace std;

int main() {
    int H, M;
    
    cin >> H >> M;

    if (M >= 45) {

        cout << H << " " << M - 45;
    }   else {

        if (H == 0) {

            cout << 23 << " " << M + 15;
        } else {

            cout << H - 1 << " " << M + 15;
        }
    }
    return 0;
}
```
먼저 시간이랑 분을 초기화 해준다. **int H,M;**

그 다음 입력을 각각 받아주고.. 

if 문을 활용해서 만약 M이 45분보다 높거나 같으면 M-45를 하게 되어있다.. 왜 이렇게 코드를 짰는가??

```
보통 분은 총 60분이다. 그래서 만약 45분에에서 45분을 빼면 0분이므로.. 딱 맞쳐서 내려간다. 하지만 40분은 어떨까?
40분을 45분으로 빼면 다시 60분을 넘어 55분이 된다.. 이러므로 우리는 이걸 방지하기 위해 딱 45분하기 크거나 같으면으로 만들어서 값이 다시 되돌아가지 않도록 만들었다.
```

그런 후 만약 M 이 45분이 아닐때는 뭘까? 한마디로 **시** 부분이 -1 된다는것이다..

그러면 H-1를 이용해서 늦출 수 있다.

하지만 시 부분이 0일때는 문제가 발생한다. 0도 다시 돌아가야되지 않는가? 23시로..

그래서 어차피 0일때는 무조건 23이 될테니.. 그냥 출력할때의 값을 23으로 정했다.. 천재적이지 않은가?? ㅋㅋ

그런 후 분 계산은 조금 신기할 것이다.. 만약 분이 45보다 작을때는 그냥 그 값에 15을 더해주면 된다.. ㅋㅋ

그러면 정확한 분 계산을 알 수 있다.

**개인적으로 수학적인 느낌이 나서 굉장히 싫은 문제였다. 하지만 그 만큼 매우 흥미로웠던 문제였다.**

## 8 백준ㅣ평균 1546번

이번 문제도 조금 어려운 편이였다 평소에 알선 평균 구하기랑은 다른 문제여서 조금 놀랬다..

그래서 구글링을 해서 답은 최대한 보지 않고서 이게 왜 이렇게 됬는지 노력할려고 했다.. 하지만 그래도 너무 어려웠다.

그래서 답을 약간 봤다. 근데 이게 무슨 일인가 내가 쓰고 있던 코드랑 비슷비슷 했다. 기분이 좋았다 ㅋㅋ..

```cpp
#include <bits/stdc++.h>

using namespace std;

int main() {
    int x;
    int max = -1;
    float hap = 0.0;
    
    cin >> x;

    int arr[x];
    
    
    
    for(int i = 0; i < x; i++)
    {
        cin >> arr[i];
        if(max < arr[i]){
            max = arr[i];
        }
    }

    for(int i = 0; i < x; i++){
        hap += (float(arr[i]) / max)*100;
    }
    
    hap /= x;
    cout << hap << endl;

    return 0;
}
 
```
처음 부터 끝까지 설명하겠다.

먼저 배열을 만들 x 값을 초기화 해준다.

그리고 최댓값을 구할 수 있는 max 변수와

합을 실수로 표현하기 위해 만든 hap 변수까지 만들었다.

그리고 x에 입력값을 받아서 입력자가 표기할 평균 과목 수를 배열에 저장할 수 있도록 공간을 늘려준다.

그런 후 int arr[x]; 를 하여 초기화를 x값 만큼 배열을 초기화 해주었다.

이제 for 문을 할 차례이다.. ㄷㄷ

일단 배열에 x만큼의 입력을 받아야겠으니 cin을 활용했다.

그런 후 if 문을 활용해서 최댓값을 구해야겠다고 생각했고, 그럴려면 각 값들을 하나씩 계산해서 max에 넣어줘야 겠다고 생각했다.

그럼 일단 최댓값은 끝이다!!

이제 각 값들에 **배열[수] / max * 100*을 해야된다. 그러기 위해서 밑에와 같이 만들었다.

```cpp
    for(int i = 0; i < x; i++){
        hap += (float(arr[i]) / max)*100;
    }
```
이런식으로 작성하면 아까와 동일한 for 문처럼 각 입력했던 값마다 값을 hap에다가 저장할 수 있다.

그런 후 마지막으로 평균 값을 구하기 위해서 **hap /= x;**을 했다.

이제 진짜 마지막으로 **cout << hap << endl;**으로 출력을 해주면 완벽히 조작된 평균을 만들 수 있다.

## 10 백준ㅣ약수 구하기 2501번

이번 문제도 머리쓰는 문제라서 어려웠다
```cpp
#include <bits/stdc++.h>

using namespace std;

int main() {
    int number1, number2, n;
    cin >> number1 >> number2;

    for(int i = 1; i <= number1; i++){
        if(number1 % i == 0){
            n++;
            if(n == number2){
                cout << i;
                return 0;
            }

        }
    }
    cout << "0";
    return 0;
}
```

이 코드를 짜면서 개인적으로 너무 힘들었다. 그래서 다른 사람들을 코드들을 참고하면서 제작하였다.

너무어려워요.....

먼저 코드를 설명하자면

약수를 찾을 변수와, 약수중 number2번째로 작은 수를 찾는 변수, 마지막으로 number2번째를 찾기 위한 보조변수 이렇게 있다.

그리고 입력값을 넣고..

for문을 활용해서  number1 만큼을 반복해주고 

if 문을 활용해서 number1 이 i로 나눠져서 0이 나오게 되면 n에 ++를 해주고

if 문을 한번 더 써서 만약 n == number2 라면 cout << i;로 number2번째에 위한 값을 찾을 수 있었습니다.

마지막으로 if 문을 통과하지 못했다면 for문을 그대로 내려와서 cout << "0"을 통해 0을 출력하는것을 볼 수 있다.

이렇게 약수를 구할 수 있는 프로그램을 만들었다.
